## CI: сборка APK в GitHub Actions

Workflow: `.github/workflows/android-apk.yml`

### Где взять APK
1) Открой вкладку **Actions** в GitHub.
2) Выбери workflow **Android APK (Buildozer)**.
3) Открой последний успешный запуск.
4) Внизу скачай Artifact **take-umbrella-apk** — внутри будет `.apk`.

### Как запустить вручную
Actions → Android APK (Buildozer) → **Run workflow**
