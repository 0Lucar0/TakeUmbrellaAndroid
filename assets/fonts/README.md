## Emoji font (Windows fix)

Kivy часто не умеет корректно отрисовывать emoji системным шрифтом на Windows (получается “квадратик”).
Решение: положить монохромный emoji‑шрифт и подключить его в приложении.

### Что сделать
1) Скачай `NotoEmoji-Regular.ttf` (Google Noto Emoji).
2) Положи файл сюда: `assets/fonts/NotoEmoji-Regular.ttf`
3) Перезапусти приложение.

После этого `☂/☔` должны отрисовываться нормально на Windows и попадут в Android-сборку.
